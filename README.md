# Reading Rascals 🦝

A reading-comprehension & phonemic-awareness practice app for grades K–8.
It runs entirely in a web browser — no installation needed.

---

## How to put it online for families (GitHub Pages — free)

You only do this once. After that you'll have a web link you can share.

### 1. Create the project on GitHub
1. Sign in at **github.com**.
2. Click the **+** in the top-right → **New repository**.
3. **Repository name:** `reading-rascals`
4. Choose **Public**, then click **Create repository**.

### 2. Upload these files
1. On the new repo page, click **"uploading an existing file"** (the blue link), or **Add file → Upload files**.
2. Open the `reading-rascals` folder on your computer.
3. Select **everything inside it** — `index.html`, all the `.jsx` files, **and** the `assets` folder — and **drag it all** into the upload box.
   - Tip: it's fine to drag the `assets` folder in as-is; GitHub keeps it together.
4. At the bottom, click **Commit changes**.

### 3. Turn on the web link
1. In the repo, click **Settings** (top menu).
2. In the left sidebar, click **Pages**.
3. Under **Branch**, choose **main** and **/ (root)**, then click **Save**.
4. Wait about a minute, then refresh. GitHub will show your live link near the top, like:
   **https://YOUR-USERNAME.github.io/reading-rascals/**
5. That's the link you share with families. 🎉

## Good to know
- **Student data stays on each device.** Names, progress, stickers, and the teacher password are saved only in that family's browser — nothing is uploaded or shared.
- **Teacher dashboard / removing readers** is protected by a password you create the first time you open it (on each device).
- **Voices** use each device's built-in text-to-speech, so they may sound a little different from computer to computer.
- **Worksheets** print with your "Your Friendly Neighborhood SLP" watermark.

Need changes? Just ask and a new version can be prepared for re-upload.
